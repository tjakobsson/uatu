# Bundled font attribution

uatu bundles **Hack Nerd Font Mono** (Regular) at
`src/assets/fonts/HackNerdFontMono-Regular.woff2` and serves it at
`/assets/fonts/HackNerdFontMono-Regular.woff2`. The vendored file is a
direct derivative of two upstream projects:

## Hack typeface
- Project: <https://github.com/source-foundry/Hack>
- License: MIT (with portions derived from the public-domain DejaVu project
  and Bitstream Vera Sans Mono under the Bitstream Vera License with
  Reserved Font Names "Bitstream" and "Vera")
- License text: [LICENSE-hack.md](./LICENSE-hack.md)

## Nerd Fonts icon patch
- Project: <https://github.com/ryanoasis/nerd-fonts>
- License: MIT
- License text: [LICENSE-nerdfonts.txt](./LICENSE-nerdfonts.txt)
